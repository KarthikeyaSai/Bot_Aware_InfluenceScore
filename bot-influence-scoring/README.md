# Bot-Aware Influence Scoring

A pipeline for social network analysis that detects bots using Graph Attention Networks (GNNs) and computes influence scores on sanitized graphs.

## Project Structure
- `data/`: Dataset storage (Cresci-2017)
- `src/`: Core logic for graph construction, GNN models, and influence scoring
- `api/`: FastAPI backend for exposing the pipeline
- `notebooks/`: EDA and prototyping
- `frontend/`: React-based dashboard
